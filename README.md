# OrangeHRM Automation Testing Framework

## Project Overview

This project is an automation testing framework developed using Python, Selenium WebDriver, and PyTest.

## Features

- Login Automation
- Add Employee Automation
- Page Object Model (POM)
- HTML Reports
- Screenshot Capture
- Reusable Driver Setup

## Technologies Used

- Python
- Selenium
- PyTest
- PyTest HTML
- VS Code

## Project Structure

pages/
tests/
utilities/
reports/

## Run the Project

```bash
pytest
```
